Название папки советую не менять, может быть некорректный сбой.
Название Html можно редактировать. 
Для размещения на сайт, откройте гланый Html Документ вашего сайта, создайте блок с помощью данного кода:
<div class="Название класса">
                        <iframe src="https://Ссылка на ваш сайт.домен сайта/Ссылка на html проекта " width="100% " height="500 " frameborder="0 "></iframe>         
                    </div>
Готово!

Пример как должно выглядить: 
div class="da-slide">
                        <iframe src="https://scskulptor.ru/Skulptor.html " width="100% " height="500 " frameborder="0 "></iframe>

</div>
